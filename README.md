# jWOPR

Simple Java API that allows you to interact with different AI providers as:

- OpenAI - ChatGPT

Quick example:

```java
    WOPR.get().ask("Who was the programmer of WOPR?"));
```

Result:

```
    The WOPR was programmed by Professor Falken in the 1983 movie WarGames.
```
